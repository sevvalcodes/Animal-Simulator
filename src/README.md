# Animal Simulator

A simple Java OOP project demonstrating inheritance, method overriding, constructors, and the `toString()` method.

## Features

- Inheritance
- Method Overriding
- Constructors
- `super` keyword
- `toString()` method

## Technologies

- Java

## Project Structure

- Animal
- Dog
- Cat
- Bird
- Fish
- AnimalSimulator

## Sample Output

Each animal displays:
- Name
- Age
- Weight
- Movement
- Sound
